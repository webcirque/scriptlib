版本更改條目

0.4 ( 開發中 )

  • 修復了頂級變量中的IO漏洞。
  • 增加了數組支持。
  • 增加了多種重載標籤頁/視窗的方法。
  • 重新定義了HTML id的含義，使得元素選擇器能夠選取所有具有相同id的元素了。
  • 添加了HTML標籤列表與語法，使得元素選擇器功能更加強大。
  • 重新加入了AJAX對象創建與發送功能。
  • 現在能夠適應MediaWiki了。
  • 現在可以使用指令來加載JS/CSS檔案了。
  • 加入了對虛數、複數、向量和平面直角坐標系的支援。

0.3

  • 改進了元素選擇器。
  • 增加了對頂級變量的IO支援。
  • 增加了對刷新頁面的支援。
  • 增加了版本選擇器。
  • 取消了對AJAX的支援。

0.2

  • 改進了元素選擇器。
  • 增加了對AJAX的簡單支援。( 不支持跨域 )

0.1

  • 增加了元素選擇器。

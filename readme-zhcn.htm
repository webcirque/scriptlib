更新日志

0.4 ( 开发中 )

  • 修复了顶级变量中的存取漏洞。
  • 添加了对数列的支持。
  • 添加了多种刷新标签页/窗口的方式。
  • 重新定义了HTML属性id，且现在支持获取所有同id的元素。
  • 增加了对HTML元素选择器的语法，且元素选择器的功能更加强大。
  • 重新加入了AJAX对象创建与发送功能。
  • 现在能勾连MediaWiki了。
  • 现在能够通过命令加载JS/CSS文件了。
  • 加入了对实数、虚数、复数、向(矢)量及对平面直角坐标系的支持。

0.3

  • 改进了元素选择器。
  • 添加了对顶级变量的IO支持。
  • 添加了对刷新页面的支持。
  • 添加了版本自动选择器。
  • 移除了对AJAX的支持。

0.2

  • 改进了元素选择器。
  • 对AJAX的简单支持。( 不支持跨域 )

0.1

  • 添加了元素选择器。
